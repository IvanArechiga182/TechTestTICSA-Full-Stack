# TechTestTICSA-Full-Stack

Repo donde está mi solución para la prueba frontend y backend aplicada por TICSA para la vacante de Desarrollador Web

# Instrucciones para correr backend

1. Clonar el repositorio
2. Entrar a la carpeta `backendTest`
3. Crear un archivo `.env` tomando como base el archivo `.env.example`
4. Instalar dependencias:
   npm install
5. Ejecutar el proyecto:
   npm run dev

# Frontend

1. Clonar repo
2. Entrar a la carpeta `frontendTest`
3. Instalar dependencias:
   npm install
4. Ejecutar:
   npm run dev
